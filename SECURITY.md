# Security Policy

## Supported Versions

The latest `0.2.x` release receives security updates.

| Version | Supported |
| ------- | --------- |
| 0.2.x   | ✅        |
| < 0.2   | ❌        |

## Reporting a Vulnerability

`@proof.com/proof-vc-web` is used in authentication flows, so please report
vulnerabilities privately rather than opening a public issue.

Email **security@proof.com** with details and reproduction steps. We aim to
acknowledge reports within a few business days and will coordinate a fix and
disclosure timeline with you.
